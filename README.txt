Pings your blog to Pingomatic using your visitors's IP's at random intervals. 

== Description ==

PPinger helps you use your visitors to ping Pingomatic.com that your blog has been updated. 

It randomizes between pinging a randomly selected amount of these RSS aggregators:

Weblogs.com
Blo.gs
Technorati
FeedBurner
Syndic8
NewsGator
MyYahoo!
PubSub.com
Blogdigger
BlogStreet
Moreover
Weblogalot
Icerocket
NewsIsFree
TopicExchange
GoogleBlogSearch
Spinn3r
Bloglines
PostRank
SkyGrid
Collecta
Superfeedr


It is a good help for getting new blogs indexed.

== Installation ==


1. Install the plugin via your WordPress installation
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Does this improve my SEO? =

Yes and no. It helps you ping your freshly created website, in fact it uses your visitors to increase exposure.
